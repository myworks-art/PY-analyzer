# Changelog

## [0.4.0] - 2026-05-21

### Добавлено

**Frontend**
- Полностью обновлён фронтенд на React + Vite.
- Добавлена multi-stage Docker-сборка (Node 20 + nginx:alpine).
- Настроен nginx для корректной отдачи статических файлов и проксирования API-запросов на backend.
- Реализованы страницы: Landing, Upload, History, Result, Docs, Help.
- Добавлены фильтры по severity и категории в списке проблем.
- Улучшена история анализов с возможностью удаления записей.
- Добавлена поддержка drag & drop и примера конфигурации.

**Backend**
- Добавлен rate limiting (slowapi) — ограничение 30 запросов в минуту на эндпоинт `/analyze/`.
- Увеличена безопасность: ограничение размера YAML-файла до 512 KB.
- Перенесён роутер анализа в `api/routers/analyze.py`.
- Улучшена обработка ошибок парсинга и валидация входных данных.
- Обновлён CORS для работы с фронтендом.

**Docker**
- Обновлён `frontend/Dockerfile` и `nginx.conf` для совместной работы фронтенда и бэкенда в Docker Compose.

### Изменено

- Обновлён `main.py` и структура API-роутеров.
- Обновлён `requirements.txt` (добавлена `slowapi`).
- Полностью заменена папка `frontend/`.

### Заменяемые файлы и папки при применении патча

- `frontend/` — полностью заменяется
- `api/routers/analyze.py` (ранее `analyze.py`)
- `main.py`
- `requirements.txt`
